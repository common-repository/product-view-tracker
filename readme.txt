== Description ==

**Product View Tracker** Tracker plugin empowers WordPress site owners to effortlessly monitor and showcase the popularity of products. Admins gain valuable analytics on which products attract the most attention, aiding in strategic decision-making. Meanwhile, customers benefit from transparency and informed purchasing decisions as they explore trending and frequently viewed items.

**Features of Product View Tracker:**

- **Product View Tracking:** Track and display the number of views for each product in a comprehensive table format.
- **Last View Timestamp:** View the timestamp of the last recorded view for detailed analysis.
- **Search Functionality:** Admins can search for specific products using an intuitive search box. Search results dynamically update as keywords are entered, facilitating quick product lookup.
- **Category Filtering:** Filter product views by selecting categories from a dropdown menu. Admins can analyze product popularity within specific categories to tailor marketing strategies.

== Installation ==

Installation is straightforward. Install the plugin directly from the WordPress plugin repository.
